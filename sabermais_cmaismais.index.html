C++ -

1. Programação Orientada a Objetos (OOP)
• Classes e Objetos: C++ permite a definição de classes, que são moldes para criar objetos. Uma classe pode conter atributos (variáveis) e métodos (funções).
• Herança: Permite que uma classe derive de outra, reutilizando código e permitindo hierarquias.
• Polimorfismo: Permite que funções ou métodos se comportem de maneira diferente dependendo do tipo do objeto que os chama.
• Encapsulamento: Mantém os dados seguros dentro de uma classe, expondo apenas o que é necessário.

2. Templates
• Permitem escrever funções e classes genéricas que podem operar com diferentes tipos de dados. Isso proporciona flexibilidade e reutilização de código.

3. Manipulação de Memória
• Ponteiros: C++ permite o uso de ponteiros para acessar diretamente a memória, o que é poderoso, mas requer cuidado para evitar vazamentos de memória.
• Referências: São outra forma de se referir a variáveis, sem a complexidade dos ponteiros, tornando o código mais seguro e legível.


4. Biblioteca Padrão (STL)
• A STL oferece containers (como vector, list, map), algoritmos e iteradores que facilitam a manipulação de dados.

5. C++ Moderno (C++11 e além)
• Lambda Expressions: Permitem criar funções anônimas diretamente onde são necessárias.
• Smart Pointers: std::unique_ptr e std::shared_ptr ajudam a gerenciar a memória automaticamente, evitando vazamentos.
